🔧 Firebase 연동 설정 가이드
1
Firebase 프로젝트 생성
console.firebase.google.com 접속 → 새 프로젝트 만들기 → 프로젝트 이름 입력 (예: lfpy-tracker)
2
Realtime Database 활성화
왼쪽 메뉴 → Realtime Database → 데이터베이스 만들기 → 테스트 모드 시작 (30일 무료) → 완료
3
보안 규칙 설정
규칙 탭에서 아래 규칙 붙여넣기 (쓰기는 허용, 읽기는 admin만)
{ "rules": { ".read": false, ".write": true, "sessions": { ".read": true, ".write": true }, "stats": { ".read": true, ".write": true } } }
4
Firebase URL 입력
데이터베이스 URL 복사 (예: YOUR_FIREBASE_URL_HERE)
5
tracker.js를 index.html에 추가
index.html의 </body> 바로 앞에 추가 (tracker.js와 같은 폴더에 두기)
<script src="tracker.js"></script>
6
tracker.js의 Firebase URL 수정
tracker.js 파일 상단의 FB_URL을 본인 URL로 교체
const FB_URL = 'https://your-firebase-url.firebaseio.com';
연결 테스트
버튼을 눌러 Firebase 연결을 확인하세요
